❌ ЕСЛИ НЕ ЗАПУСКАЕТСЯ:

ОШИБКА 1: "Gradle sync failed"
Решение:
  └─ File → Sync Now (Ctrl+Shift+I)
  └─ Или нажмите на кнопку "Retry"

ОШИБКА 2: "java.lang.UnsupportedClassVersionError"
Решение:
  └─ File → Project Structure
  └─ Project → SDK → выберите Java 17+
  └─ Нажмите OK

ОШИБКА 3: "Port 8080 already in use"
Решение:
  └─ Отредактируйте .env
  └─ SERVER_PORT=8081
  └─ Перезагрузите приложение

ОШИБКА 4: "Connection refused" к БД
Решение:
  └─ Убедитесь PostgreSQL запущен
  └─ Или используйте Supabase (видеть SUPABASE_SETUP.md)
