🔄 ДАННЫЕ FLOW:

Android App
    ↓
Firebase Auth (получить ID Token)
    ↓
HTTP запрос с Bearer token к Backend
    ↓
FirebaseTokenFilter верифицирует
    ↓
SecurityContext сохраняет user ID
    ↓
Controller обрабатывает запрос
    ↓
Service обращается к Репозиторию
    ↓
Repository обращается к БД (через JPA)
    ↓
PostgreSQL/Supabase возвращает данные
    ↓
Response отправляется клиенту
