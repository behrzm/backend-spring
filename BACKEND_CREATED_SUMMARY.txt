🎓 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ
═════════════════════════════════════════════════════════════

Получить профиль:
   curl -H "Authorization: Bearer TOKEN" \
     http://localhost:8080/api/v1/profiles/me

Добавить XP:
   curl -X POST \
     -H "Authorization: Bearer TOKEN" \
     -H "Content-Type: application/json" \
     -d '{"deltaXp": 150}' \
     http://localhost:8080/api/v1/profiles/me/xp

Завершить уровень:
   curl -X PUT \
     -H "Authorization: Bearer TOKEN" \
     -H "Content-Type: application/json" \
     -d '{
       "language": "python",
       "track": "beginner",
       "levelId": 1,
       "stars": 3,
       "completed": true
     }' \
     http://localhost:8080/api/v1/levels/progress

❓ TROUBLESHOOTING
═════════════════════════════════════════════════════════════

Ошибка: "Connection refused"
   → Подождите 10 секунд после docker-compose up
   → Проверьте: docker-compose ps

Ошибка: "401 Unauthorized"
   → Убедитесь, что используете "Bearer YOUR_TOKEN"
   → Проверьте, что токен не истёк

Ошибка: "Port 8080 already in use"
   → Измените port в application.yml

Полный troubleshooting: backend-spring/README.md
